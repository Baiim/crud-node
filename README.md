# CRUD (Create, Read, Update, Delete) with Node.Js + Express.Js + Mysql Database.
## Getting Started
this is my first node.Js project using the express.Js module and also using MYSQL as the database.

### Installing

```
npm install
```
or

```
yarn install 

```

### How to run project

```
npm run dev
```

or

```
yarn run dev
```
